# Making a Custom Icon Set
## File Stucture
- (MultiMC Root Folder)
  - iconthemes
      - custom
        - scalable
          - (Your Custom Icons)
        - index.theme

### Example index.theme
```
[Icon Theme]
Name=(Theme Name)
Comment=(Theme Comment)
Inherits=multimc
Directories=scalable

[scalable]
Size=48
Type=Scalable
MinSize=16
MaxSize=256
```
